Practising Dreamweaver
     

1. Try out Dreamweaver

Go to Macromedia website and click TRY on the first page. You can download Dreamweaver MX 2004 for a free 30-day trial while waiting for your own copy of Dreamweaver.
After downloading you can also try out Dreamweaver's very own tutorials from Help > Tutorials.
 

2. Borrow Library books

National Library branches have some very good books on Dreamweaver. The reference code is 005.72 in the Non-Fiction Adult Section.
