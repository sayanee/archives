Creating Layers

 up
 
  • Open up the Layers Palette
  • Click the icon
  • New Layer is added on top of the selected layer
  
     

Merging Layers

 up
 
  • Select the top layer out of all the layers you want to merge
  • Click in the empty box beside the picture of the layers below the top layer
    A chain icon will appear
  
     

Adding Fill layer or Adjustment Layer

 up
 
  • Click the icon
  • Choose the appropriate option
 
     

Locking Layers

 up
 
  • The Lock portion of the Layers Palette is used
  • Click to lock tranparent pixels in the layer
  • Click to lock image pixels in the layer
  • Click to lock layer's position
  • Click to lock all the available locks
