File Menu

 up
 


  • Open as Library: Opens the library of other flash files so that their library items can be used.
  • Import: Transfers graphics, videos and sounds to the library of the Flash file
  • Publish Settings: A Flash file can be converted to a graphic file, html file or a window projector file
  
     

Edit Menu

 up
 

  • Paste in Place: Pastes a copied oject in the same place as it was before.
  • Preferences: You can edit some of the preseting here
  
     

View Menu

 up
 


  • Rulers ,Grids and Guides are used for designing purposes
  
     

Insert Menu

 up
 

  • Convert to Symbol: to create a graphic, button or a movie clip from an existing object on the stage
  • New Symbol: to create a graphic, button or a movie clip from no existing objects on the stage
  • Layer: adds a new Layer on top of the selected layer
  • Layer Folder: Adds a Layer folder on top of the selected layer
  • Motion Guide: Adds a motion guide layer to the selected layer
  • Frame: adds another frame to the right of the selected frame
  • Remove Frame: deletes the frame selected
  • Keyframe: Adds a keyframe to the frame selected
  • Blank Keyframe: Adds a blank keyframe
  • Clear Keyframe: Removes the keyframe
  • Create Tween Motion: Flash automatically creates a tween between 2 keyframes
  • Scene: Adds a new scene
