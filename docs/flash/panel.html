Tools

 up
 
  • Tools panel is used to draw and edit graphics on the stage

  
   

Timeline

 up
 
  • Timeline shows the layers and sequences of an animation
     

Properties

 up
 
  • Properties Panel changes as different options are chosen from the Tools Panel
     

Align

 up
 
  • Align is used places several objects in an organised fashion
  
     

Color Mixer

 up
 
  • Color Mixer is used to create a gradient color that is used to fill an object.
  • The gradient can be solid, linear, radial or a bitmap
  • After creating a gradient you can Add to Swatch so that the gradient can be used again
    Click the icon in the Color Mixer Panel
    Select Add Swatch and the color will be added to a list of colors in the Color Swatch Panel
  
     

Color Swatch

 up
 
  • Shows a list of pre-defined colors

  
     

Info

 up
 
  • Shows the information about the color and position ( in terms of x and y) when the mouse moves over any object.
  
     

Scene

 up
 
  • Shows the list of scenes the flash file has
  • Scenes can be duplicated by the icon
  • New scenes can be added by the icon
 
     

Transform

 up
 
  • Transform Panel can rotate, enlarge or skew an object with numerical values.
    It has the same function as the Free Transform Tool which transforms an object without any numerical values.
  
     

Actions

 up
 
  • Actions Panel contains all the action script codes added to a frame, button or a movie clip

  
     

Movie Explorer

 up
 
  • Movie Explorer shows information about the flash files
  • It has organised list of texts, buttons, movie clips, graphics, scenes, actions , sounds, layers and videos.
  
     

Library

 up
 
  • Library has a list of all the items that the flash file needs and all the items imported to the flash file.
  • Movie Clip
  • Bitmap
  • Symbol
  • Sound
  • Video
  • Buttons
  • Folder
  
     

Common Library

 up
 
  • Common library has a storage of buttons and sounds which can be used.
