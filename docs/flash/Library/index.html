PRACTISING PHOTOSHOP
     

1. Try out Photoshop

  

Go to Adobe website and choose Adobe Photoshop Tryout. You can download Photoshop CS for a free 30-day trial while waiting for your own copy of Photoshop.

     

2. Borrow Library Books

  

National Library branches have some very good books on Photoshop. The reference code is 006.6869 in the Non-Fiction Adult Section.
