Publishing Formats

 up
 

Flash movies can be converted to many other formats:

  • .swf        Flash file is created and it needs a Flash Player to run
  • .html       HTML page is created where flash writes its own html codes for us
  • .gif          This format can be used if we want to create a simple animated graphic
  • .jpeg       For creating static graphic files
  • .png        For creating static graphic files whose background is transparent
  • .exe        This format does not need the Flash Player and it can run on its own
  
     

Export Movies

 up
 

Flash can also convert your animation to a video or movie file with the the file extension .avi.
This move file can then run in the Windows Media Player.

  • Go to File> Export Movie
