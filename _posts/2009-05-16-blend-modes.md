2. Dissolve
3. Multiply
4. Screen
5. Overlay
6. Soft Light
7. Hard Light
8. Color Dodge
9. Linear Dodge
10. Color Burn
11. Linear Burn
12. Hard Mix
13. Divide
14. Addition
15. Subtract
16. Difference
17. Darken Only
18. Lighten Only
19. Hue
20. Saturation
21. Color
22. Luminosity
