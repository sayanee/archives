title: Whether vs. If
---

This is an interesting one i thought. I am not conscious when to use 'whether' and when to use 'if'

- Whether - for a choice/alternatives
- If - for a condition

Note: 'whether or not' is a redundant phrase and saying 'whether' is enough!
