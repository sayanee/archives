Found another way of blocking spam sites / pop-up ads without using any pop-up blocking/heavy anti-virus softwares. Here's how...
<ol>
	<li>In Windows, go to the directory...
<font color="#800000">C:\WINDOWS\system32\drivers\etc</font></li>
	<li>Locate the <a href="http://en.wikipedia.org/wiki/Hosts_file">hosts file</a></li>
	<li>Open the host file using the notepad</li>
	<li>Add the domain name we want to block in this format:
<font color="#800000">127.0.0.1         www.************.com</font></li>
	<li><font color="#000000">Save and close the file</font></li>
</ol>
